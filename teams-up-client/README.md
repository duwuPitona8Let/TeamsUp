# Teams-Up Client — Angular приложение

Клиентское приложение для системы управления командами **Teams-Up**.

## Технологический стек

- **Angular 21** (standalone компоненты)
- **TypeScript**
- **SCSS** для стилизации
- **RxJS** Signals для управления состоянием

## Возможности

- **Аутентификация**: регистрация, вход, JWT токены
- **Управление командами**: просмотр, создание, редактирование, удаление
- **Управление пользователями**: просмотр списка, профили, редактирование ролей (для админов)
- **Назначения**: добавление и удаление участников команд
- **Ролевая модель**: разделение доступа для User, Project Manager, Admin

## Установка и запуск

### Требования

- Node.js >= 18
- npm >= 9
- Запущенный backend (teams-up) на http://localhost:3000

### Установка зависимостей

```bash
npm install
```

### Команды

```bash
# Запуск в режиме разработки
npm run start

# Сборка проекта
npm run build

# Запуск сервера для просмотра собранного приложения
npx http-server dist/teams-up-client/browser
```

## API конфигурация

Приложение подключается к backend по адресу: `http://localhost:3000`

Для изменения адреса обновите `apiUrl` в сервисах:

- `src/app/services/auth.service.ts`
- `src/app/services/user.service.ts`
- `src/app/services/team.service.ts`
- `src/app/services/assignment.service.ts`

## Маршруты

| Маршрут           | Описание               | Доступ         |
| ----------------- | ---------------------- | -------------- |
| `/login`          | Страница входа         | Все            |
| `/register`       | Страница регистрации   | Все            |
| `/dashboard`      | Главная панель         | Авторизованные |
| `/teams`          | Список команд          | Авторизованные |
| `/teams/create`   | Создание команды       | PM, Admin      |
| `/teams/:id`      | Детали команды         | Авторизованные |
| `/teams/:id/edit` | Редактирование команды | PM, Admin      |
| `/users`          | Список пользователей   | Авторизованные |
| `/users/:id`      | Профиль пользователя   | Авторизованные |
| `/admin`          | Админ панель           | Admin          |

## Роли пользователей

- **user** — обычный пользователь
- **project_manager** — менеджер проектов (может создавать команды, назначать участников)
- **admin** — администратор (полный доступ, включая удаление команд и изменение ролей)
